Bank Service Тема проекта "Банковская система" - управление банковскими операциями, клиентами, счетами, картами и финансовыми транзакциями.

Основные сущности

Базовые сущности:

"Customer" - клиенты банка (личная информация)
"Account" - банковские счета (расчетные, сберегательные)
"Card" - банковские карты (дебетовые, кредитные)
"Transaction" - финансовые транзакции и переводы
Бизнес-сущности:

"Loan" - кредитные
"Deposit" - вклады
"ExchangeRate" - курсы валют
"CreditCardApplication" - заявки на кредитные карты
Возможности сервиса

Базовые CRUD операции:

"Управление клиентами" - регистрация, просмотр, обновление, удаление
"Управление счетами" - открытие, закрытие, пополнение, блокировка
"Управление картами" - выпуск, блокировка, замена, активация
"История операций" - просмотр и управление транзакциями
Бизнес-операции:

"Кредитование" - оформление и погашение кредитов
"Вкладные операции" - открытие/закрытие депозитов с начислением процентов
"Конвертация валют" - обмен по актуальным курсам с переводом на другой аккаунт.
"Кредитные карты" - подача заявок и процесс одобрения
"Процентные начисления" - автоматическое начисление на сберегательные счета
"Денежные переводы" - между счетами с проверкой баланса
